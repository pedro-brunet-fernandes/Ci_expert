Projetos em SystemVerilog

Este repositório contém diferentes projetos desenvolvidos em SystemVerilog, organizados em pastas separadas. Cada projeto possui sua própria estrutura, incluindo descrição do circuito (DUT), testbench e instruções de simulação.

Estrutura
hello/: projeto introdutório utilizando TCL para demonstrar execução de scripts e conceitos básicos da linguagem.
projeto_ph/: projeto em SystemVerilog contendo a implementação e simulação de circuitos digitais (como portas lógicas), seguindo o fluxo de EDA com Icarus Verilog.
Organização

Cada pasta é independente e contém todos os arquivos necessários para execução do respectivo projeto. Para rodar um projeto, basta acessar sua pasta e seguir as instruções descritas no README específico.
